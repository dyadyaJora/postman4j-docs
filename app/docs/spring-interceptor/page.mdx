# Spring Interceptor

## Installation

In Gradle:
```gradle
implementation group: 'dev.jora.postman4j', name: 'spring-interceptor', version: '0.0.4'
```

In Maven:
```xml
<dependency>
    <groupId>dev.jora.postman4j</groupId>
    <artifactId>spring-interceptor</artifactId>
    <version>0.0.4</version>
</dependency>
```
